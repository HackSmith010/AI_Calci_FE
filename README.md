## 🔧 Installation  

### 1️⃣ Clone the Repository  
```sh
git clone https://github.com/HackSmith010/AI_Calci_FE.git
cd AI_Calci_FE
```

### 2️⃣ Install Dependencies
```sh
npm install
```

### Start the Development Server
```sh
npm start
```
### Build for Production
```sh
npm run build
```
